Running

I run to eat more cookies. I run to get away from the frustrations of building a web portfolio. I run to explore the town. I run to have fun.

I began running before I started college because all of my friends were doing it. I continued to run because I enjoy challenging myself and pushing myself to do better. I believe that anyone who wants to run can be a runner! I may not be fast, but I will continue to get better.
